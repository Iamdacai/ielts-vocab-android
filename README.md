# 📘 雅思智能背单词 - Android App

## 🎯 项目概述
这是一个专注于雅思词汇学习的移动应用，采用**本地优先**的设计理念，确保用户在无网络环境下也能完整使用所有核心功能。

## ✨ 核心功能

### 📚 词汇学习系统
- **281个精选雅思词汇**：覆盖学术和通用场景
- **智能间隔重复算法**：基于SM-2算法的个性化复习计划
- **学习进度跟踪**：实时统计掌握度、学习天数等指标
- **离线可用**：所有词汇数据本地存储，无需网络

### 🎤 发音练习系统
- **高质量TTS发音**：集成OpenClaw TTS服务
- **发音评分**：基于语音识别的发音质量评估
- **跟读练习**：支持录音和回放对比
- **音标显示**：标准IPA音标标注

### ⚙️ 个性化设置
- **每日学习计划**：自定义新词学习数量（默认20词/天）
- **复习时间设置**：设置最佳复习时间段
- **学习目标管理**：设定总词汇掌握目标

## 🏗️ 技术架构

### 前端技术栈
- **React Native**: 0.71.14
- **Expo**: 48.0.21
- **React Navigation**: 底部标签导航 + 堆栈导航
- **SQLite**: 本地数据库存储

### 核心服务
- `vocabulary.js`: 词汇数据管理
- `learningProgress.js`: 学习进度跟踪
- `spacedRepetition.js`: 间隔重复算法
- `pronunciation.js`: 发音服务集成
- `pronunciationAssessment.js`: 发音评分

### 数据库设计
- **words表**: 存储词汇基础信息
- **user_progress表**: 存储用户学习进度
- **user_config表**: 存储用户个性化设置

## 🚀 快速开始

### 环境准备
```bash

# 启动开发服务器
npm start

# 运行Android应用
npm run android
```

